### Key Bugs Found
- **Podman secrets with trailing newlines**: Multiple secrets (`osmen-redis-password`, `osmen-postgres-password`) had `\n` appended. Redis accepted it but downstream clients (Nextcloud, Paperless) couldn't match. Fix: `printf '%s' "$PASSWORD" | podman secret create --replace`.
