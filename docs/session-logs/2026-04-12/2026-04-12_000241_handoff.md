## Decisions Made

1. **Homebrew for Node instead of nvm**
   - nvm removed entirely (broke OpenClaw doctor paths, took 2.3 GB)
   - Homebrew node@24 is pinned and stable; OpenClaw systemd unit path is deterministic
   - Impact: PATH now sourced from `/home/linuxbrew/.linuxbrew/bin`; `.bashrc` updated

2. **nomic-embed-text for local embeddings**
   - Replaces any cloud embedding provider for memory search
   - Provider: Ollama (already running); model: `nomic-embed-text`
   - Impact: memory search works offline, no external API key needed

3. **Plex as native .deb (not Podman quadlet)**
   - NTFS external drives (sda2, sdb2, sdc1) are complex to bind-mount through Podman
   - `user-osmen-media.slice` provides cgroup coverage for transcoding processes
   - No OsMEN-OC quadlet needed; Plex data dir stays at `/var/lib/plexmediaserver`

4. **P2.3 Tailscale deferred**
   - Not needed for current local-only install; can be added later
   - Task ID 1 remains open as deferred
